Quick Sizing Guide
Landing Opening Width Calculation
📏 Center-opening Door
Opening Width = (Hoistway Width - 100) ÷ 2, Round down to nearest 50mm
📏 Side-opening Door
Opening Width = (Hoistway Width - 225) ÷ 1.5, Round down to nearest 50mm

With 30mm Aluminum Hoistway Frame: Hoistway Internal = Hoistway External − 60

All dimensions based on 2200mm structural car height
